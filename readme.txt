Simple Twitter timeline synchronization with your Posts Archive

== Description ==

Great tool to archive your tweets for better SEO & show your tweets on your web with no backward limit at posts with special category.

 Functionality overview:
 
- Archival and display of Twitter feed history anywhere on a Worldpress website
- Tweets become part of Worldpress posts and because they are archived, they appear in search which is great for search engines and increases your site SEO
- Default WordPress templates twentyten and twentyeleven (or any other template) will show the post format as “status”
- Frequency of synchronization with Twitter can be adjusted as needed.
- Tweets that are deleted in Wordpress are not imported again.


Showcase/Tutorial on YouTube http://www.youtube.com/watch?v=vtmtAB2C3mE


== Installation ==

1. Upload `twitter2posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your local timezone in WordPress Settings/General before setting-up the plugin
4. Go to Plugin Settings, write your twitter username, then click to Save & Sync
5. Plugin will create special category and imports last 20 tweets (Twitter API limitation). Every new tweet will be imported automatically.

If you want to see how to setup plugin, see tutorial on YouTube: http://www.youtube.com/watch?v=vtmtAB2C3mE
